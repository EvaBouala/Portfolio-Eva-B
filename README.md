## Apercu

Ce projet est une vitrine moderne et responsive qui met en avant :

- Mon identité et mon parcours
- Mes projets avec pages detaillees
- Mes competences techniques
- Un espace de contact

## Stack technique

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Composants UI reutilisables

## Installation locale

1. Installer les dependances

```bash
npm install
```

2. Lancer le serveur de developpement

```bash
npm run dev
```

3. Ouvrir le site

```text
http://localhost:3000
```

## Production

```bash
npm run build
npm run start
```
